# Release Notes: Case Entry Tracking Redesign

## What's New ✨
- Case entry tracking now **resets at the start of each month** — giving you a fresh look at how the current month is going
- A **trend line** shows whether things are improving or getting worse as the month goes on
- **Department breakdown** so you can see exactly where cases are being missed
- **Year-to-date percentage** for a bigger-picture view across all of 2026

## Improvements 🔧
- The tooltip now stays open when you hover over it, so you can interact with the trend chart
- Hover any point on the trend line to see the exact value for that day
