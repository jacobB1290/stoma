# Release Notes: Redesigned Case Entry Tracking

## What's New ✨
- Case entry tracking now **resets monthly** instead of rolling 90 days — fresh start each month
- Smart summary paragraph explains the data in plain language, adapting to the actual numbers
- **Trend line** shows how the miss rate is moving through the month — hover any point to see the value
- **Department breakdown** shows where gaps are concentrated
- **Year-to-date** section with color-coded percentage for long-term reference

## For Users 👤
- Hover the staff-entered pill to open the dashboard — it stays open so you can interact with the trend chart
- The summary text types in with a smooth fade animation, highlighting key numbers in bold
- The trend line is interactive — hover over any day to see the cumulative miss rate at that point
- "General" department is now labeled "Digital"
- Stats reset on the 1st of each month so you can track monthly progress
